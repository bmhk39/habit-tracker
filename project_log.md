# 習慣トラッカー プロジェクトログ

## プロジェクト概要

**アプリの目的**: 習慣化継続支援。モチベーション。やるべき習慣、積み上げてきた実績、夢への進歩の見える化。

**思想**: 「全ては習慣。継続は力なり。1万時間の法則。」

**設計方針**: UX優先・柔軟な記録
- **習慣化を支援するツール**。厳しさよりも「続けやすさ」を重視。
- **時間計測機能**で「見えない努力」を可視化。累計時間が積み上がる達成感。
- **手動修正OK**。過去の記録も自由に編集可能。完璧主義にならない。

**開発姿勢**:
- **抜本的見直しを恐れない**。まだプロトタイプ段階であり、使用者は自分だけ。
- 既存の仕様や構造に囚われず、より良いアイデアがあれば積極的に再構築する。
- シンプルさを追求し、複雑化を避ける。
